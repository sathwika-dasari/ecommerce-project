# Ecommerce Project

This is a full stack ecommerce web application.

## Technologies Used
- React (Frontend)
- Flask (Backend)
- SQLite Database

## Features
- View products
- Add to cart
- Checkout system
- Order placement
- API integration

## Project Structure

backend/
- controllers
- routes
- models
- config

frontend/
- components
- pages
- services

## How to Run

Backend:
cd backend
python app.py

Frontend:
cd frontend
npm install
npm run dev
